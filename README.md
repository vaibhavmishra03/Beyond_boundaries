A modern, animated farewell website for college cricket friends, built with React, Vite, and Tailwind CSS.

## Features
- **Hero Section**: Dynamic "Aurora" background with "Lightning" text effects.
- **Timeline**: Scroll-triggered animation of memories.
- **Scoreboard**: Animated stats counters (Matches Played, etc.).
- **Gallery**: Staggered animation entry with Lightbox and Zip Download.
- **Guestbook**: Leave messages (persisted locally).
- **Music Player**: Floating pulse button with specific song support.
- **Privacy First**: Consent modal included.

## Quick Start

1.  **Install Dependencies**
    ```bash
    npm install
    ```

2.  **Run Locally**
    ```bash
    npm run dev
    ```

3.  **Build**
    ```bash
    npm run build
    ```

## Customization

- **Images**: Add photos to `src/assets` and update `src/data/galleryImages.js`.
- **Events**: text/dates in `src/data/timelineEvents.js`.
- **Music**: Update URL in `src/components/MusicPlayer.jsx`.

## Deployment

This site is verified for zero-config deployment on Vercel or Netlify.
Simply connect your repository and deploy.
